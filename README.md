# JSS - jjuidev Stack Skills

Claude Code agent skills for JavaScript/TypeScript development.

## Installation

```bash
npx skills add jjuidev/jss
```

## Available Skills

- **react** - React development patterns and best practices
- **next** - Next.js development (coming soon)
- **rn** - React Native development (coming soon)
- **nest** - NestJS development (coming soon)

## Usage

Skills install to `~/.claude/skills/jss/`. Claude Code automatically loads SKILL.md files.

## Structure

```
jss/
├── SKILL.md           # Root manifest
├── skills/
│   ├── react/         # React patterns
│   ├── next/          # Next.js patterns
│   ├── rn/            # React Native patterns
│   └── nest/          # NestJS patterns
└── docs/
```
